Bug Test Generator – Options

API keys

Used when you select models with provider “OpenAI”. This must be an OpenAI API key (from platform.openai.com), not just a ChatGPT Plus/Pro subscription.
Used when you select models with provider “Google Gemini”. Get this from Google AI Studio → API Keys (aistudio.google.com). :contentReference[oaicite:4]{index=4}

Models (right-click submenu)

These entries appear under “Generate test page” when you right‑click a selection. Each entry is:
  • Menu label – what you see in the context menu.
  • Model ID – the provider’s model ID (e.g. gpt-4.1 or gemini-2.5-flash).
  • Provider – OpenAI or Google Gemini.
